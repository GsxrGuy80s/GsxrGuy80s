# 🛠️ Building the Sovereign Brain
    Engineering high-performance, local-first AI infrastructure to eliminate the "Cloud Tax."
    
    ## The Mission
    I build systems that treat privacy as a feature, not an afterthought. My work focuses on moving heavy AI workloads off the cloud and onto your hardware.
    
    ## Current Focus
    * **Pocket Node:** High-performance mobile inference (Android/NDK).
    * **PrivateLLM:** The local-first interface for your private data.
    * **ZCT Stack:** The "OS" for the modern homelab.
